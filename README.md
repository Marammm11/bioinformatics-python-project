#  DNA Sequence Analyzer (Python)

##  Project Overview
This project is a simple bioinformatics tool developed in Python to analyze DNA sequences.  
It performs essential sequence analysis tasks commonly used in molecular biology and bioinformatics.

---

##  Objectives
- Apply programming skills to biological data
- Understand basic DNA sequence analysis
- Simulate simple bioinformatics workflows

---

##  Features
-  Nucleotide counting (A, T, G, C)
-  GC content calculation (%)
-  DNA → RNA transcription
-  Reverse complement generation
-  Input validation for DNA sequences

---

##  Example
**Input:**
ATGCGTACGTTAGC

**Output:**
Nucleotide count: {'A': 3, 'T': 4, 'G': 4, 'C': 3}
GC Content: 50.0 %
RNA: AUGCGUACGUUAGC
Reverse complement: GCTAACGTACGCAT

##  How to Run the Program

1. Make sure you have Python installed (Python 3 recommended)
2. Download or clone this repository
3. Run the script:


python dna_analyzer.py


4. Enter a DNA sequence when prompted

##  Biological Background
DNA sequences are composed of four nucleotides: Adenine (A), Thymine (T), Guanine (G), and Cytosine (C).

- **GC Content** is an important parameter in genomics, affecting DNA stability.
- **Transcription** converts DNA into RNA by replacing Thymine (T) with Uracil (U).
- **Reverse complement** is used in sequence alignment and analysis.
